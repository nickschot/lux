### Installation

```bash
npm install -g lumen-framework
```

### Creating Your First Project

Use the new command to create your first project.

```bash
lumen new <app-name>
```

### Generators

Lux allows you to use the CLI to generate boilerplate for the following types:

- `model`
- `controller`
- `serializer`
- `middleware`
- `migration`
- `resource`
- `util`

```bash
lumen generate <type> <name> [attrs...]
```

### Running

To run your application use the serve command.

```bash
cd <app-name>
lumen serve
```
