# social-network

## Installation

*   `git clone https://github.com/<this-repository>`
*   `cd social-network`
*   `npm install`

## Running / Development

*   `lumen db:reset`
*   `lumen db:migrate`
*   `lumen db:seed`
*   `lumen serve`

## Testing

*   `lumen test`
